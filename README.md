# SafeCode

SafeCode is a zero-trust licensing and IP-protection platform based on hosted execution, short-lived entitlements, and auditable policy controls.

## Monorepo (Sprint 1 Foundation)

```text
apps/
  api-gateway/
packages/
  shared-contracts/
infra/
  migrations/
githooks/
```

## Getting Started

Requirements:
- Node.js 20+

Commands:
- `npm install`
- `npm run verify`

## Sprint 1 deliverables

- Workspace scaffold and CI baseline
- API Gateway auth context module
- Core PostgreSQL schema migration
